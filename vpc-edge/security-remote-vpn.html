Security Awareness of the Remote VPN in VPC Edge Server

The VPC Edge Access Point V1.0.0 has several ways to implement Remote User VPN, including the OpenVPN, IPsec and SAML.

VPN Settings Saved in Server:

Security IPSec OpenVPN SAML IPSec SAML OpenVPN
Token(Password) Plain Hash Plain Hash
Token Expired Never Never 1 hour 1 hour

The way to change the token or password of the remote VPN:

IPSec or OpenVPN: Change the password from the web site UI.

SAML: Click the "Generate" button to generate a new token. Any token will expire in 1 hour.

A VPN connection is still alive even if the SAML token is expired, until the endpoint user ends the existing connection, then the next connection will use the new token.
